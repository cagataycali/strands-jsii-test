    API Reference¶

    Complete API reference for the Strands Agents jsii SDK.

    Core Classes¶

    StrandsAgent¶

    The main agent class that orchestrates model calls and tool execution.

    class StrandsAgent:
    def __init__(self, config: AgentConfig)
    def invoke(self, prompt: str) -> AgentResponse
    Method Returns Description
    invoke(prompt) AgentResponse Run the agent loop with a prompt

    AgentConfig¶

    Configuration for creating an agent.

    class AgentConfig:
    def __init__(
        self,
        model: ModelProvider,
        system_prompt: str = "",
        tools: List[ToolDefinition] = [],
        max_cycles: int = 50,
        conversation_manager: ConversationManager = None,
        callback_handler: CallbackHandler = None,
    )
    Parameter Type Default Description
    model ModelProvider required LLM provider
    system_prompt str "" System instructions
    tools List[ToolDefinition] [] Available tools
    max_cycles int 50 Max agent loop iterations
    conversation_manager ConversationManager None Message management strategy
    callback_handler CallbackHandler None Lifecycle event handler

    AgentResponse¶

    Response from an agent invocation.

    class AgentResponse:
    text: str           # Final text response
    input_tokens: int   # Input tokens used
    output_tokens: int  # Output tokens generated

    ToolSpecification¶

    class ToolSpecification:
    def __init__(self, name: str, description: str, input_schema: str)
    Parameter Type Description
    name str Tool name (used by model)
    description str What the tool does
    input_schema str JSON Schema string for parameters

    ToolDefinition (Abstract)¶

    class ToolDefinition:
    def __init__(self, spec: ToolSpecification)
    def execute(self, input_json: str) -> str  # Override this

    class ContextAwareToolDefinition:

    CallbackHandler (Abstract)¶

    class CallbackHandler:
    def on_model_start(self, messages_json: str) -> None
    def on_model_end(self, response_json: str) -> None
    def on_model_error(self, error: str) -> None
    def on_tool_start(self, tool_name: str, input_json: str) -> None
    def on_tool_end(self, tool_name: str, result: str) -> None
    def on_tool_error(self, tool_name: str, error: str) -> None
    def on_agent_start(self, prompt: str) -> None
    def on_agent_end(self, response_json: str) -> None
